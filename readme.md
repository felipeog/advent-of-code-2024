# Advent of Code 2024

https://adventofcode.com/2024

## Tests

- `.` is the `advent-of-code-2024` folder
- Verbose output with `-v` flag

Run all tests:

```zsh
go test ./...
```

Run tests in a folder:

```zsh
go test ./day01
```

Run specific test in a folder:

```zsh
go test ./day01 -run TestFirstHalf
```
